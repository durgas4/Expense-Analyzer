# AI-Assisted Expense Analyzer


This application allows users to upload a CSV file containing financial transactions, categorize the expenses, and visualize the data. It provides insights into spending habits and generates a bar chart representing expenses by category.

## Features
- **File Upload**: Users can upload a CSV file with columns: Date, Description, Amount.
- **Expense Categorization**: Transactions are categorized based on keywords in the description.
- **Visualization**: A bar chart displays total spending per category.
- **AI Insights**: Insights about spending habits are generated.
